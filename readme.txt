The RecentlyPlayedStore is a Java program that provides an in-memory store for recently played songs for a user.
It can accommodate N songs per user, with a fixed initial capacity. 
The store has the capability to store a list of song-user pairs, with each song linked to a user. 
It can fetch recently played songs based on the user and eliminate the least recently played songs when the store becomes full.
Steps to run the project
Clone the repository to the required ide using git --clone or pull command.
Exeecute the main class in the src folder.
